## 各回の30分構成（確定版 第1〜5回）

### 第1回 セキュリティ（11枚）
0-15分 スライド → 15-21分 実演 → 21-26分 質疑 → 26-30分 振り返り

### 第2回 環境（13枚）
0-15分 スライド → 15-23分 実演 → 23-28分 質疑 → 28-30分 振り返り

### 第3回 基本ガード（12枚）
0-15分 スライド（CLAUDE.md→Permission→Sandbox→Hooks→階段→統合settings.json）→ 15-23分 実演 → 23-28分 質疑 → 28-30分 振り返り

### 第4回 自動化・共有（12枚）
0-15分 スライド（Skills→MCP→MCP→Skills変換→コマンド&Plugin→コンテキストエンジ）→ 15-23分 実演 → 23-28分 質疑 → 28-30分 振り返り

### 第5回 AIとの仕事の型（12枚）
0-2分 前回振り返り → 2-4分 やりがちなこと → 4-6分 5つの防止アクション → 6-8分 要件（SDD/AIDLC）→ 8-9分 ドキュメント → 9-10分 計画 → 10-12分 テスト&レビュー → 12-13分 成熟度モデル → 13-15分 実演導入 → 15-23分 実演 → 23-28分 質疑 → 28-30分 振り返り
